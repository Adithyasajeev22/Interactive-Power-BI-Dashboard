# 📊 Interactive Power BI Dashboards

This repository contains a collection of interactive Power BI dashboards developed to transform raw data into meaningful business insights. These projects demonstrate expertise in data visualization, data modeling, DAX, Power Query, and dashboard design to support data-driven decision-making.

---

## 🚀 Projects Included

### 1. Home Appliances Analysis Dashboard

An interactive dashboard designed to analyze sales performance, profit, product categories, payment modes, and regional trends in the home appliances sector.

#### Key Features
- Sales and profit analysis
- Product-wise performance metrics
- Category-wise sales comparison
- Regional sales distribution
- Payment mode analysis
- Year-over-year category trends
- Interactive charts and KPI cards

#### Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## 🎬 Netflix Analysis Dashboard

#### Key Features

- Content growth trends over time
- Director count by genre
- Rating analysis
- Interactive visualizations and filters

#### Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization
